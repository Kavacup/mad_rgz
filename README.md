# MAD_RGZ

## Что это

В репе представлена Расчетно-графическая работа по МАД 2025. Ведется работа с датасетом "California_Houses_15_col_dot.csv", пробуются различные ансамбли для построения модели регрессии. Отчет представлен в книжке **.ipynb*.

## Как работать

Для быстрого развертывания используется `conda`, настройки окружения приведены в *environment.yml*.
Для этого достаточно вызвать `conda env create -f environment.yml`.
Список окружений - `conda env list`
Сменить окружение - `conda activate <name>`
